Signal Strength¶

class signalStrength.SignalNode(MACAddress, name, signalStrength)[source]¶

A highly optimized object to keep track of the signal strength associated with a node. It is hashable, so it can be a key in a dict or an element in a set.

MACAddress and name are immutable once the instance is created, but signalStrength can be changed

MACAddress[source]¶

Get the name of the node, as a string.

Example: OLIN_WH

name[source]¶

Get the name of the node, as a string.

Example: OLIN_WH

signalStrength.getAvgSignalNodes(samples=3, tsleep=0.15)[source]¶

Returns list of signal nodes

signalStrength.getAvgSignalNodesDict(samples=3, tsleep=0.15)[source]¶

Get the average signal strength from samples samples, at intervals of tsleep seconds (although it will take a bit longer since getCoords() takes a while to execute)

Returns a dictionary with key : value pairs of the form MAC_Address : [Signal_Strength, Network_Name] Example:

{
‘00:20:D8:2D:2C:C1’: [14, ‘OLIN_CC’], ‘00:20:D8:2D:B3:C0’: [12, ‘OLIN_CC’], ‘00:20:D8:2D:65:02’: [12, ‘OLIN_WH’], ‘00:20:D8:2D:85:40’: [38, ‘OLIN_CC’]

}

signalStrength.getSignalNodeDict()[source]¶

Scans (or gets cached versions, on some systems) of wireless signal strengths around the computer, using platform-dependent methods. Returns a dict mapping SignalNode identifiers to SignalNodes
