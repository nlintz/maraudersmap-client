API¶

class api.Coordinate(x, y, w, distance=0)[source]¶

An optimized way of representing an x,y,w location and (optionally) a distance.

Parameters:
  • x (int) – x coordinate in pixels, on image w
  • y (int) – y coordinate in pixels, on image w
  • w (int) – There are two images of maps in the original marauder’s map. This attribute specifies to which map the x,y coordinates refer, 1 or 2.
  • distance (int) – Not entirely sure what this is for...

Warning

It is not possible to add attributes to this object

class api.Location(encodedName, coordinate)[source]¶

An object to represent the combination of a place name and a coordinate.

Parameters:
  • encodedName (str) – The server-side representation of the name of the place. Ex: WH,in,rm309
  • coordinate (Coordinate) – The x,y,w coordinate of the place.
getReadableName()[source]¶

Parses the encodedName and converts it into a human-readable string.

This is useful for menu entries, for example.

Warning

This doesn’t work properly; it is from the original code.

api.do_cloak(username)[source]¶

Tell the server to stop displaying a user with username on the map. If an Update happens, the user will reappear on the map again.

This should require authentication and should only be possible for the active user.

Parameters:username (str) – Username of user to conceal on the map.
Returns:a tuple (serverSuceededBOOL,serverResponseSTR)
