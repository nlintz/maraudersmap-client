Map Client¶

The main component of the client-side Marauder’s Map Application

The progam’s interface is accessible via a drop-down menu from an icon in the system tray. It displays the user’s estimated location, provides access to a preferences window

Tab for advanced preferences in the PreferencesWindow.

class mapclient.GeneralPrefs[source]¶

Tab for general preferences in the PreferencesWindow.

class mapclient.GetLocationThread[source]¶

A task to obtain the nearest location.

Tasks are used by the PreferencesWindow, which adds them to the global thread pool as necessary. When a task is complete, it emits a location_updated_signal, which is received by the PreferencesWindow.

class mapclient.NewLocationThread[source]¶

A task to let the user create a new bind.

It spawns the bind creation view of the marauder’s map after getting a signal reading.

class mapclient.PreferencesWindow[source]¶

The preferences window is the owner of everything else in the program. It should not be instantiated more than once.
